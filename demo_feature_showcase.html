Feature Overview

The new DISABLE_DEMO_MODE environment variable provides an easy way to force the SI-HQ application into production mode, helping users identify which areas are fully functional without demo data.

Usage Examples

1. Enable Production Only Mode

# .env file
DISABLE_DEMO_MODE=true
FIREBASE_PROJECT_ID=your-real-project
GOOGLE_APPLICATION_CREDENTIALS=./real_service_account.json
# ... other real Firebase config

2. Check Status via API

GET /api/system/demo-mode

Response:
{
  "demo_mode": false,
  "demo_mode_disabled": true,
  "firebase_available": true,
  "database_connected": true,
  "message": "Production mode enabled (demo disabled)"
}
